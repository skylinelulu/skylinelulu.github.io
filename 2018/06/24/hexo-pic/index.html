1.如何在博客中插入图片？参考文章

（1）把主页配置文件_config.yml 里的post_asset_folder选项设置为true
（2）在hexo目录下执行命令：npm install hexo-asset-image –save，即下载安装可以上传本地图片的插件hexo-asset-image
（3）再运行hexo n “xxx”来生成md博文时，/source/_posts文件夹内除了xxx.md文件还有一个同名的文件夹
